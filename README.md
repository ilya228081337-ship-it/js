# Защитники Япоши

Игра-симулятор работы в японском магазине, где нужно правильно обслуживать разных типов клиентов.

## Развертывание на Netlify

### Шаг 1: Подготовка репозитория

1. Создайте новый репозиторий на GitHub
2. Инициализируйте git в проекте (если еще не сделано):
```bash
git init
git add .
git commit -m "Initial commit"
```

3. Добавьте удаленный репозиторий и загрузите код:
```bash
git remote add origin YOUR_GITHUB_REPO_URL
git branch -M main
git push -u origin main
```

### Шаг 2: Развертывание на Netlify

1. Зайдите на [netlify.com](https://netlify.com) и войдите через GitHub
2. Нажмите "Add new site" → "Import an existing project"
3. Выберите "GitHub" и найдите ваш репозиторий
4. Настройки развертывания (должны заполниться автоматически):
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
5. В разделе "Environment variables" добавьте переменные окружения:
   - `VITE_SUPABASE_URL`: значение из вашего .env файла
   - `VITE_SUPABASE_ANON_KEY`: значение из вашего .env файла
6. Нажмите "Deploy site"

### Шаг 3: После развертывания

После успешного развертывания:
- Netlify автоматически создаст URL вида `https://your-site-name.netlify.app`
- Каждый push в main ветку будет автоматически обновлять сайт
- Вы можете настроить собственный домен в настройках Netlify

## Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшн сборки
npm run preview
```

## Игровые механики

- Выберите персонажа (Ира, Сашка или Полинка)
- Обслуживайте клиентов правильно для получения очков
- У каждого клиента свой характер и требования
- Управляйте кассой - она увеличивается при правильных ответах
- Некоторые персонажи имеют особые способности

## Технологии

- React + TypeScript
- Vite
- Tailwind CSS
